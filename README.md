## Projetos

### 1. AutiTarefas
Aplicativo voltado para crianças com TEA (Transtorno do Espectro Autista) e Síndrome de Down, com:

- Tarefas interativas de habilidades de vida diária  
- Gamificação para tornar o aprendizado divertido  
- Acompanhamento de desempenho  
- Feedback em tempo real  
- Recursos de acessibilidade
